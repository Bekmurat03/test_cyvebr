Magic Token

📋 Тапсырма

Сервер сессия деректерін role_token атты Cookie-де сақтайды. Бұл Cookie – ішінде JSON бар, Base64 кодталған жол.

Мысалы: eyJ1c2VyIjogImFsaSIsICJyb2xlIjogImd1ZXN0In0=

Ағымдағы рөліңіз: guest.
Cookie-ді өзгертіп рөлді admin етіп жаз, сонда /admin бетінде Flag табасың!

💡 Кеңес

1. Браузерде F12 → Application → Cookies → role_token
2. Мәнін алып Base64 декодта (python немесе CyberChef)
3. JSON-дағы "role": "guest" → "role": "admin" деп өзгерт
4. Жаңа мәнді Base64 кодтап Cookie-ге қой
5. /admin бетіне кір

🐍 Python декодтау мысалы
import base64, json
# Декодтау
cookie = "СІЗДІҢ_COOKIE_МӘНІ"
data = json.loads(base64.b64decode(cookie))
print(data) # {'user': '...', 'role': 'guest'}

# Жаңа admin cookie жасау
data['role'] = 'admin'
new_cookie = base64.b64encode(json.dumps(data).encode()).decode()
print(new_cookie)
