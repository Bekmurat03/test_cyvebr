📋 Тапсырма

challenge.wav аудио файлы жай ән емес. Оның ішінде ZIP мұрағаты жасырылған. ZIP-тің ішінде flag.txt файлы бар. Тап!

⬇️ challenge.wav жүктеу
💡 Кеңес

WAV файлының соңына бинарлық деректер қосылған. ZIP сигнатурасы: 50 4B 03 04.

Файлды hex-тен ізде немесе binwalk / foremost қолдан. Немесе Python-мен ZIP offset-ін тауып шығарып ал.

🐍 ZIP шығарып алу (Python)
ZIP_SIG = b'\x50\x4B\x03\x04'

with open('challenge.wav', 'rb') as f:
    data = f.read()

# ZIP сигнатурасының орнын тап
offset = data.find(ZIP_SIG)
if offset == -1:
    print("ZIP табылмады!")
else:
    print(f"ZIP табылды: offset={offset}")
    zip_data = data[offset:]
    with open('extracted.zip', 'wb') as out:
        out.write(zip_data)

# Шығарып алу
import zipfile
with zipfile.ZipFile('extracted.zip') as z:
    z.extractall('output')
# output/flag.txt файлын оқы
🛠️ Терминал командалары
# binwalk (Linux/Mac)
binwalk -e challenge.wav

# foremost
foremost -i challenge.wav -o output_dir

# unzip тікелей байқап көру
unzip challenge.wav -d output_dir
